This is the README file.

This is a collection of the programs related to our Robcup Junior Premier Rescue Project.

/Testing - Contains all the programs written during the testing of this project
/Main - Contains all the programs written to be run on the robot for the competition
/Setup - Contains all the programs to be run on the day for the purposes of calibration and aspects of testing
/Other - Contains other files such as Data from calibration and testing
